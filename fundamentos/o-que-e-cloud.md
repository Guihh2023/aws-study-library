# ☁️ O que é Cloud Computing

## O que é?

Cloud Computing é o fornecimento de serviços de tecnologia pela internet, permitindo acesso a servidores, armazenamento, bancos de dados, redes e softwares sem precisar possuir infraestrutura física própria.

---

## Para que serve?

Permite:
- armazenar arquivos
- hospedar aplicações
- criar servidores
- escalar sistemas
- reduzir custos com infraestrutura

---

## Como funciona?

Os recursos ficam em datacenters de provedores cloud, como a AWS.

O usuário acessa esses recursos pela internet pagando apenas pelo uso.

---

## Principais vantagens

- Escalabilidade
- Alta disponibilidade
- Redução de custos
- Segurança
- Flexibilidade

---

## Exemplo prático

Hospedar um site utilizando Amazon S3 ou EC2.

---

## Boas práticas

- Monitorar custos
- Configurar permissões corretamente
- Utilizar autenticação MFA

---

## Erros comuns

- Achar que cloud elimina todos os custos
- Não controlar permissões de acesso
- Não monitorar recursos ativos

---

## Resumo rápido

Cloud Computing = uso de serviços de tecnologia pela internet sob demanda.
