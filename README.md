# ☁️ AWS Study Library

Biblioteca de estudos AWS com documentação simples, prática e didática para iniciantes em Cloud Computing.

O objetivo deste repositório é documentar meus estudos de forma simples, organizada e prática, ajudando também outras pessoas iniciantes na AWS.
